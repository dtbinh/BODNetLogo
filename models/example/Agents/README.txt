###############################################################################################################################
###############################################################################################################################

BODNetLogo lap files are very similar to traditional lap files, with only two differences.

1. You can now use 'true' or 'false' in the place of a sense name for a sense that you want to always pass (or fail)
2. Frequency should be specified in ticks not seconds or minutes.
