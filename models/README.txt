The default structure for models should be as follows:


Information about the model:
	../models/[modelName]/README.txt
	
bnconfig file
	../models/[modelName]/[modelName].bnconfig
	 
nlogo file
	../models/[modelName]/[modelName].nlogo 

agent lap file(s)
	../models/[modelName]/Agents/[AgentPluralName].lap
