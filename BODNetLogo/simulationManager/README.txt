This is a readme file for part of the BODNetLogo program, please note that BODNetLogo, whilst fully functional,
is not a finished program, you may find bugs or errors that are not handled properly and the user interface is not

contact the author at contact@mhbrooks.com

This is the directory for the simulationManager code


###############################################################################################################################
###############################################################################################################################

SimManager is called by the BODNetLogo class in the main package, it controls the various aspects of setting up a BODNetLogo simulation

LoadConfigFile is the parser for .bnconfig files that can be read to pre-populate the user interface with the simulation details
